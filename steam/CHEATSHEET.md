## To run game

### Game option
for proton game, mostly
- select game, right click > **Properties...**
- in **General** add **LAUNCH OPTIONS** `LD_PRELOAD="" gamemoderun %command%`

### Native Linux game
- select compatibility to `Steam-Play-None`
